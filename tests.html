File Definitions

yara

YARA is an open-source tool used to identify and classify malware by creating rules based on textual or binary patterns. In Black Swan, YARA powers the rule-based detection engine.

engine.c

The core C file responsible for scanning files, applying YARA rules, and reporting detections. Optimized for speed and low resource usage.

gui.py

Python script implementing the graphical user interface using Tkinter. Handles user input, scan controls, and displays results.

rules.yar

YARA rule file containing custom detection rules for various malware signatures and behaviors.
YARA rules are signature-based detection patterns that help identify malware—viruses, ransomware, Trojans—by matching specific strings, hex patterns, or binary signatures against files or memory samples.
rule ExampleRule
{
strings
$a="text here"
$b={E2 34 A1 C8 23 FB}
condition
$a or $b
}
